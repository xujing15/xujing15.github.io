hexo+github搭建博客第二篇

hexo设置多标签：tags: [标签1,标签2,标签3]

添加rss：

1.安装RSS插件：npm install hexo-generator-feed --save
2.在\hexo_config.yml添加
feed:
type: atom
path: atom.xml
limit: 20
3.在\hexo\themes\concise_config.yml添加
rss: /atom.xml
4.如果已经有RSS一栏，则不用执行此步。在\hexo\themes\concise\layout_partial\header.ejs中，找到ul添加一行代码。
5.将\hexo_config.yml中的url添加为自己的网址。
url: http://xujing15.github.io

添加sitemap

1.安装sitemap插件：npm install hexo-generator-sitemap --save
2.在\hexo_config.yml添加
sitemap:
path: sitemap.xml
3.访问xujing15.github.io/sitemap.xml

3.在要使用文章导航的文章首部加入toc: true
